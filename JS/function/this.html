    this 关键字

    JeasonDeng 2017-3-3 this

    参考 (opens new window)

    JavaScript 语言之中，一切皆对象，运行环境也是对象，所以函数都是在某个对象之中运行
    this 就是函数运行时所在的对象（环境）
    this 的指向是动态的，没有办法事先确定到底指向哪个对象

    # this 对象

    解析器在调用函数时, 每次都会向函数内部传递两个隐含的参数 —— this & arguments

    • this 指向的是一个对象 (函数执行的上下文对象)
    • 根据函数调用方式的不同, this 会指向不同的对象
      • 以函数的形式调用时, this 永远都是 window
      • 以方法的形式调用时, this 就是调用方法的对象
    var A = {
  name: '张三',
  describe: function () {
    return '姓名：'+ this.name;
  }
};

var name = '李四';
var f = A.describe;
f() // "姓名：李四"   全局调用，this 是 window

    # this 不会向上继承

    如果 this 所在的方法不在对象的第一层，这时 this 只是指向当前一层的对象，而不会继承更上面的层

    var a = {
  p: 'Hello',
  b: {
    m: function() {
      console.log(this.p);
    }
  }
};

a.b.m() // undefined

    # 数组遍历 API 的回调中不要使用 this

    数组遍历 API 的回调中的 this 是 window，不是当前数组对象

    var o = {
  v: 'hello',
  p: [ 'a1', 'a2' ],
  f: function f() {
    this.p.forEach(function (item) {
      console.log(this.v + ' ' + item);
    });
  }
}

o.f()

    # 改变 this 的方法

    JavaScript 提供了 call、apply、bind 这三个方法，来切换/固定this的指向

    # call()

    函数实例的 call 方法，可以指定函数内部 this 的指向

    call 方法的参数，应该是一个对象。如果参数为空、null 和 undefined，则默认传入全局对象
    如果 call 方法的参数是一个原始值，那么这个原始值会自动转成对应的包装对象，然后传入 call 方法

    var obj = {};

var f = function () {
  return this;
};

f() === window // true
f.call(obj) === obj // true


var f = function () {
  return this;
};

f.call(5)
// Number {[[PrimitiveValue]]: 5}

    # apply()

    apply 方法的作用与 call 方法类似 区别 —— 传参形式不同

    // call
func.call(thisValue, arg1, arg2, ...)

// apply 第二个参数是一个数组
func.apply(thisValue, [arg1, arg2, ...])

    # bind()

    bind 方法用于将函数体内的 this 绑定到某个对象，然后返回一个新函数 bind() 传参形式与 call() 一样

    func.bind(thisValue, arg1, arg2, ...)
    1

    区别

    • bind() 返回一个新的改变 this 指向的函数，需要调用返回的新函数才会执行
    • call() 会立即调用当前函数
    var add = function (x, y) {
  return x * this.m + y * this.n;
}

var obj = {
  m: 2,
  n: 2
};

var newAdd = add.bind(obj, 5);
newAdd(5) // 20

    # 总结 this

    调用形式 this 指向
    全局调用 window
    对象方法调用 调用方法的对象
    new 构造函数形式调用 新创建的那个对象
    使用 call() 和 apply() 时 指定的那个对象(就是第一个参数)
    在事件响应函数 触发事件的 dom 对象
    箭头函数中 其外层作用域的 this

    Foo() 和 new Foo() 的区别 —— 前者 this - window, 而后者 this - 实例对象
