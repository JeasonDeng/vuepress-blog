    ScrollReveal 页面滚动插件

    JeasonDeng 2017-3-9 页面滚动

    scrollReveal.js 也是一款页面滚动显示动画的 JavaScript，能让页面更加有趣，更吸引用户眼球

    • 兼容PC端和移动设备
    • 不依赖其他任何文件

    scrollReveal (opens new window)

    使用参考 (opens new window)

    # 安装

    npm install scrollreveal
    1

    或者

    <script src="https://unpkg.com/scrollreveal"></script>
    1

    # 引入

    <!-- 在 head 中引入-->
<head>
    <script src="https://unpkg.com/scrollreveal"></script>
</head>
    1
    2
    3

    # 使用

    <h1 class="headline">
    Widget Inc.
</h1>
<p class="tagline">
    The perfect widgets.
    <span class="punchline">Forever.</span>
</p>

<script>
ScrollReveal().reveal('.headline');
ScrollReveal().reveal('.tagline', { delay: 500 });
ScrollReveal().reveal('.punchline', { delay: 2000 });
</script>

    # 配置

    默认配置 (opens new window)

    {
    delay: 0,
    distance: '0px',
    duration: 600,
    easing: 'cubic-bezier(0.5, 0, 0, 1)',
    interval: 0, // 同时进入视口的元素间隔多少时间显示（多用于水平的一系列元素）
    opacity: 0,
    origin: 'bottom', // 初始方位（相对于目标位置）
    rotate: {
        x: 0,
        y: 0,
        z: 0,
    },
    scale: 1,    
    mobile: true,
    reset: false, // 每次离开视口是否重置
    afterReset: function (el) {},
    afterReveal: function (el) {},
    beforeReset: function (el) {},
    beforeReveal: function (el) {},
}

    # 闪现问题

    在页面首次加载时，需要动画的元素会有闪现

    解决 (opens new window)

    • 给动画元素添加类 load-hidden(类名可以自定义)
    html.sr .load-hidden {
    visibility: hidden;
}
